uint32_t reshade::api::descriptor_set_update::array_offset

Array index in the specified binding at the specified offset. Only meaningful in Vulkan, in OpenGL and other APIs this has to be 0 (since each GLSL array element gets a separate binding index).

◆ binding

uint32_t reshade::api::descriptor_set_update::binding

◆ count

uint32_t reshade::api::descriptor_set_update::count

Number of descriptors to update, starting at the specified offset. If the specified binding has fewer than count array elements starting from array_offset, then the remainder will be used to update the subsequent binding starting at array element zero, recursively.

◆ descriptors

const void* reshade::api::descriptor_set_update::descriptors

Pointer to an array of descriptors to update in the set (which should be as large as the specified count). Depending on the descriptor type this should be pointer to an array of buffer_range, resource_view, sampler or sampler_with_resource_view.
